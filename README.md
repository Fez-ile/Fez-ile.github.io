#  Fezile Amanda Mkhonza – Full Stack Developer Portfolio

---

##  Project Overview

This is my personal portfolio website built to showcase my skills, projects, and professional experience as a Full Stack Developer.

The application is designed with a strong focus on:

* Clean UI/UX design
* Responsive layouts
* Interactive frontend behavior
* Structured and maintainable code

---

##  Key Technical Highlights

This is not just a static site, it includes multiple interactive and dynamic features:

###  Frontend Interactivity

* Custom **typewriter effect** for dynamic role display
* Smooth scrolling navigation
* Active section highlighting on scroll
* Scroll-triggered animations using **Intersection Observer API**

###  UI/UX Enhancements

* Fully responsive design (mobile-first adjustments)
* Animated hover effects across components
* Sticky navigation with dynamic shadow behavior
* Scroll-to-top floating action button
